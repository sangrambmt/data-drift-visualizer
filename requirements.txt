# Data Drift Visualizer

**Detect data drift before your model fails.**

One command → clear risk → visual proof.

---

## What makes this different

Most drift tools are:

* heavy
* hard to set up
* impossible to explain

This is:

* **1 command**
* **human-readable output**
* **visual + statistical combined**

---

## ⚡ 30-Second Demo

### Run

```bash
python -m driftviz.cli examples/baseline.csv examples/current.csv
```

### Output

```
Overall Drift: MEDIUM (0.638)
Recommendation: Monitor closely

Top Risk Columns:
  income (HIGH)
  age (MEDIUM)
  country (MEDIUM)

income → HIGH (0.821)
  - distribution shifted (PSI high)

age → MEDIUM (0.563)
  - moderate shift detected
```

---

### Visual Report

```bash
open reports/report.html
```

Includes:

* column-level drift
* explanations
* histograms (baseline vs current)

---

## What it actually does

Compares:

```
training data (baseline)
vs
production data (current)
```

Detects:

| Type        | Method       |
| ----------- | ------------ |
| Numerical   | KS Test, PSI |
| Categorical | Chi-square   |

Then converts:

```
raw stats → drift score → risk label → explanation
```

---

## Why this matters

Data changes silently.

Models don’t.

Result:

* predictions degrade
* alerts come too late
* debugging becomes reactive

This tool makes it **proactive**.
